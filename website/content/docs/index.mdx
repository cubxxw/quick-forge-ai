# 文档

这是 Quick Forge AI 的文档页面，您可以在这里了解如何使用我们的产品。

## 快速入门

要开始使用 Quick Forge AI，请按照以下步骤操作：

1. 安装
2. 配置
3. 运行

## 安装

```bash
npm install quick-forge-ai
```

## 配置

创建一个 `config.js` 文件：

```javascript
// config.js
export default {
  apiKey: 'your-api-key',
  model: 'gpt-3.5-turbo'
};
```

## 运行

```javascript
import { QuickForge } from 'quick-forge-ai';
import config from './config.js';

const app = new QuickForge(config);
app.start();
```
